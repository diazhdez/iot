Sistema de Control de Enfriadores de Caldera con Arduino

El proyecto de "Sistema de Control de Enfriadores de Caldera con Arduino" se enfoca en desarrollar una solución automatizada y versátil para controlar los enfriadores (ventiladores) de una caldera de manera eficiente y segura. Utilizando la plataforma Arduino y una combinación de módulos, como Ethernet, LCD y relé, este proyecto ofrece una solución integral para el monitoreo y control de los enfriadores.

El objetivo principal de este proyecto es proporcionar una interfaz intuitiva y confiable que permita a los operadores supervisar el estado de los enfriadores, activarlos cuando sea necesario y generar informes sobre el rendimiento del sistema. Con esta solución, se busca mejorar la eficiencia operativa, reducir los costos de mantenimiento y garantizar un funcionamiento seguro y confiable de la caldera en entornos industriales.

Materiales:

-Arduino: Placa microcontroladora programable
-Aplicación Arduino: Entorno de desarrollo integrado (IDE) para programar Arduino
-Módulo Ethernet: Permite la comunicación del sistema con una red
-Módulo Relé: Controla el encendido y apagado de dispositivos eléctricos
-Módulo LCD: Muestra información en una pantalla LCD
-2 Ventiladores de 120 V: Enfrían la caldera

Objetivos:

-Incorporar la capacidad de generar informes periódicos sobre el rendimiento del sistema, incluyendo datos como la frecuencia de activación de los enfriadore
-Utilizar un módulo Ethernet para permitir la conectividad en red y la comunicación remota con el sistema, lo que facilita el monitoreo y control desde ubicaciones remotas.
-Desarrollar una interfaz de usuario intuitiva y fácil de usar que permita a los operadores supervisar el estado de los enfriadores, activarlos manualmente si es necesario
